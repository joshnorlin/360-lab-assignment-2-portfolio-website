Selected Projects

Below is a brief selection of practical, student‑friendly projects. Each item emphasizes simplicity, clear structure, and a focus on learning by building. I aim to keep scopes small, test ideas quickly, and document steps so others can follow along and adapt the approaches.

  • Job Application Tracker

    This project helps track positions, deadlines, and notes in one place. The goal is to practice CRUD patterns, basic validation, and organized data views. I focused on a clean layout, minimal friction for data entry, and simple filters that highlight actionable items like upcoming due dates.

  • Simple To‑Do List

    A lightweight to‑do app that supports adding tasks, checking them off, and removing items. It demonstrates state updates and usability basics such as keyboard navigation and clear affordances. The app favors a straightforward UI and predictable behaviors to reduce cognitive load.

  • Student Club Event Page

    An accessible events page showing meetings and activities with dates, locations, and descriptions. I practiced semantic HTML, responsive sections, and careful typography. The design keeps information scannable while remaining visually consistent with the rest of the site.
